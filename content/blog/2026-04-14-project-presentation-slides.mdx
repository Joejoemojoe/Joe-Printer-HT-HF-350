# Project Presentation Slides

This update packages a slide style presentation using the content already in the docs and update posts. Each slide is written as a short, scannable block for the feed.

## Slide 01 - Title

**Joe Printer HT HF 350**

High flow CoreXY build log, documentation, and tuning reference.

## Slide 02 - Why this build

- Enclosed, rigid CoreXY platform for repeatable results
- Documentation first approach: everything lives in MDX
- Focus on calibration discipline, not just speed

## Slide 03 - Architecture snapshot

- CoreXY motion with a Z moving gantry
- 2020 extrusion frame and linear rails
- Klipper firmware for macros and input shaping

Reference: [Voron 2.4 Style CoreXY Printer Overview](/docs/voron-24-style-printer)
